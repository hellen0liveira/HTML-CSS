Principais formatações

Negrito/destaque

Nesta frase, temos um termo em destaque usando a tag STRONG. (Semântica)

Itálico/ênfase

Nesta frase, temos um termo em ênfase usando a tag EM. (Semântica)

Texto marcado

Nesta frase, temos um texto marcado utilizando a tag MARK.

Texto letras pequenas

Nesta frase, temos um texto em letras pequenas usando a tag SMALL.

Texto deletado e texto inserido

aqui temos um texto deletado usando a tag DEL. Ele indica que o texto deve ser lido mas não considerado. (Semântica)

aqui temos um texto inserido usando a tag INS. Indica ênfase e que ele foi inserido depois. (Semântica)

Texto sobrescrito

usado para colocar elemento em cima, usando a tag SUP entre o elemento que quer elevar. Como x ao quadrado, ex: x2

Texto subscrito

usado para colocar elemento em baixo, usando a tag SUB entre o elemento que quer colocar em baixo. Como em: H2O
